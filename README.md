# 🪪 Telegram Info Bot

A simple public Telegram bot that shows user ID, forwarded message ID, channel ID, and contact ID.

---

## 🤖 Features
- Show your Telegram ID
- Show username & language
- Get sender ID from forwarded messages
- Get channel ID from forwarded posts
- Get contact ID
- Auto reply to messages

---

## ⚙️ Requirements
- Python 3.9+
- Telegram Bot Token from @BotFather

---

## 📦 Install

```bash
git clone https://github.com/your-username/telegram-info-bot
cd telegram-info-bot
pip install -r requirements.txt
